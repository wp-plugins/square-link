Ajoutez et gérez facilement vos emplacements publicitaires Square Link sur votre site.

== Description ==

Grâce au plugin Square Link, vous pouvez insérer et gérer facilement des emplacements publicitaires sur votre site. 
Pour ajouter un emplacement sur votre site, vous avez plusieurs possibilités :

* Dans un zone dynamique grâce au widget 
* Dans n'importe quel contenu (pages, article, ...) grâce au bouton disponible dans le WYSIWYG

Aucune connaissance technique n'est requise pour installer les emplacements sur votre site. Renseignez simplement l'identifiant Square Link de votre site dans le menu "Réglages" > "Square Link" de votre interface d'administration. Pour connaître l'identifiant de votre site, rendez-vous sur la [page d'installation de vos site](https://app.square-link.com/publisher/sites "page d'installation de vos site")

Pour toute question ou information complémentaire, n'hésitez pas à nous contacter à hello@square-link.com

Pour utiliser ce plugin, vous devez être inscrit sur [Square Link](http://www.square-link.com "Square Link")

== Installation ==

**Pour installer le plugin :**

* Via FTP
1. Uploader le dossier "square-link" dans "/wp-content/plugins/",
2. Activer le plugin "Square Link" dans l'onglet "Extensions installées" de votre interface d'administration Wordpress

* Via l'administration Wordpress
1. Dans l'interface d'administration de votre site, choisir l'onglet "Extensions" puis "Ajouter",
2. Rechercher le mot "Square Link",
3. Sous le plugin "Square Link" cliquer sur "Installer Maintenant",
4. Activer le plugin en cliquant dans l'onglet "Extensions installées"

**Pour utiliser le plugin :**

1. Renseignez l'identifiant de votre site dans l'onglet "Réglages" > "Square Link"., 
2. Ajoutez un emplacement via le widget en sélectionnant l'emplacement souhaité dans la liste déroulante., 
3. Ajoutez un emplacement dans n'importe quel contenu en cliquant sur le bouton "Ajouter un emplacement Square Link" au dessus de votre zone de contenu.

== Frequently Asked Questions ==

= Comment créer un compte Square Link =

Pour utiliser ce plugin, il est nécessaire d'avoir au préalable créé un compte Square Link. Pour créer gratuitement votre compte, allez sur la page suivante  : https://app.square-link.com/register/

= Comment connaître l'identifiant de mon ? =

Vous pouvez retrouver facilement l'identifiant de votre site. Pour cela, rendez-vous sur la page suivante : https://app.square-link.com/publisher/sites (vous devez être connecté à votre compte Square Link)
