Ajoutez et gérez facilement vos espaces Square Link sur votre site Wordpress via les widgets et les shortcodes.

== Description ==

Grâce au plugin Wordpress de Square Link, vous pouvez insérer et gérer facilement des espaces sur votre site internet. 
Pour ajouter un espace sur votre site, vous avez plusieurs possibilités :

* Utiliser le widget Square Link : indiquez simplement un titre et l'identifiant de l'espace à afficher
* Insérer un shortcode sous la forme [squarelink id="xx"] en remplacant xx par l'identifiant de l'espace à afficher

Pour retrouver facilement les identifiants de vos espaces publicitaires à insérer, rendez-vous sur la [page d'installation de vos espaces](https://app.square-link.com/publisher/sites/installation "page d'installation de vos espaces")

Pour toute question ou information complémentaire, n'hésitez pas à nous contacter via la [page d'aide](https://app.square-link.com/help "page d'aide")

Pour utiliser ce plugin, vous devez être inscrit sur [Square Link](http://www.square-link.com "Square Link")

== Installation ==

Pour installer le plugin :

* Via FTP
1. Uploader le dossier "square-link" dans "/wp-content/plugins/" sur votre serveur via FTP
2. Activer le plugin "Square Link" dans l'onglet "Extensions installées" de votre interface d'administration Wordpress

* Via l'administration Wordpress
1. Dans l'interface d'administration de votre site, choisir l'onglet "Extensions" puis "Ajouter"
2. Rechercher le mot "Square Link"
3. Sous le plugin "Square Link" cliquer sur "Installer Maintenant"
4. Activer le plugin en cliquant dans l'onglet "Extensions installées"

Pour utiliser le plugin :

* Via Widget : dans l'onglet "Apparence" puis "Widgets" de l'administration
* Via Shortcode : en insérant le shortcode [squarelink id="xx"] en remplacant xx par l'identifiant de votre espace
