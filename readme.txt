Ajoutez et gérez facilement vos emplacements Square Link sur votre site Wordpress via les widgets et les shortcodes.

== Description ==

Grâce au plugin Wordpress de Square Link, vous pouvez insérer et gérer facilement des emplacements sur votre site internet. 
Pour ajouter un emplacement sur votre site, vous avez plusieurs possibilités :

* Widgets
* Shortcodes

Aucune connaissance technique n'est requise pour installer les emplacement sur votre site. Renseignez simplement l'identifiant Square Link de votre site dans la menu "Réglages" > "Square Link" de votre interface d'administration WordPress. Pour connaître l'identifiant de votre site, rendez-vous sur la [page d'installation de vos site](https://app.square-link.com/publisher/sites "page d'installation de vos site")

Pour toute question ou information complémentaire, n'hésitez pas à nous contacter via la [page d'aide](https://app.square-link.com/help "page d'aide")

Pour utiliser ce plugin, vous devez être inscrit sur [Square Link](http://www.square-link.com "Square Link")

== Installation ==

Pour installer le plugin :

* Via FTP
1. Uploader le dossier "square-link" dans "/wp-content/plugins/" sur votre serveur via FTP
2. Activer le plugin "Square Link" dans l'onglet "Extensions installées" de votre interface d'administration Wordpress

* Via l'administration Wordpress
1. Dans l'interface d'administration de votre site, choisir l'onglet "Extensions" puis "Ajouter"
2. Rechercher le mot "Square Link"
3. Sous le plugin "Square Link" cliquer sur "Installer Maintenant"
4. Activer le plugin en cliquant dans l'onglet "Extensions installées"

Pour utiliser le plugin :

* Renseignez l'identifiant de votre site dans l'onglet "Réglages" > "Square Link".
* Ajoutez un emplacement via les widget en sélectionnant l'emplacement souhaité dans la liste déroulante.
* Ajoutez un emplacement dans n'importe quel contenu en cliquant sur le bouton "Ajouter un emplacement Square Link" au dessus de votre zone de contenu.
